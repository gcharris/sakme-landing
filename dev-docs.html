Key Design Decisions

Architectural choices that shape the system.

REASONING

Society of Thought

Multi-variant tournament generation outperforms single-shot prompting. SOT-001 experiment: 40% peak capture rate. 3 variants, 0.2 wildcard ratio, [0.7, 0.9] temperature range.

GOVERNANCE

Narrative Governor

Deterministic kill switches (Arc Wholeness, Character Integration, Setup-Payoff Distance) because LLMs cannot reliably self-govern. N1-N5 violation taxonomy with severity routing.

KNOWLEDGE

Three-Layer Codex

Sovereign > Project > External trust hierarchy. Director-blessed knowledge takes precedence. Crystallization promotes raw material to trusted knowledge with blessing scores.

CONSISTENCY

Transactional Outbox

Atomic writes across PostgreSQL + Neo4j + EventStore without 2PC. OutboxRelay processes pending entries asynchronously with retry logic and HITL approval gates.

LLM ACCESS

BYOK Architecture

Users supply their own API keys. Platform provides the intelligence layer (soul + state + policy). BudgetOrchestrator skips budget checks when is_byok=True.

BILLING

Provider-Agnostic ABC

BillingService interface with MockBillingService and PaddleBillingService adapters. Add a new provider by implementing one ~150 LOC adapter file.
