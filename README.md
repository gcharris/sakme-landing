# sakme-landing

Landing page for [sakme.ai](https://sakme.ai) — a creative workspace that remembers.

Deployed via GitHub Pages with custom domain.

## Setup

1. Enable GitHub Pages (Settings → Pages → Deploy from branch: `main`, folder: `/ (root)`)
2. Add custom domain: `sakme.ai`
3. Configure DNS: CNAME record pointing `sakme.ai` → `<username>.github.io`
